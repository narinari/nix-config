#+title: My Nix Configurations

* Highlights
- [[https://nixos.wiki/wiki/Flakes][Nix Flakes]]
- [[https://github.com/nix-community/home-manager][home-manager]]
- Encrypt secret resources with sops-nix and age
- Multi-platform(Linux & Darwin(MacOS)), Multi-architecture(x86_64 & aarch64)
- Sub-flakes for independent lock files (infra / workstations)

* Architecture

#+begin_example
nix-config/
  flake.nix              # Orchestrator: deploy-rs, devShell, re-exports
  flake.lock             # deploy-rs, pre-commit-hooks, path: refs only

  infra/
    flake.nix            # Home servers (nixpkgs = nixos-unstable)
    flake.lock           # Independent lock file

  workstations/
    flake.nix            # Desktop + Darwin (nixpkgs = master)
    flake.lock           # Independent lock file

  hosts/                 # Shared host configurations (plain .nix)
  home-manager/          # Shared home-manager configurations
  modules/               # Custom NixOS/Darwin modules
  lib/                   # Helper functions
  overlays/              # Package overlays
  pkgs/                  # Custom packages
#+end_example

* Commands
** Install nix
- Disable channel by default
- Enable the use flake

#+begin_src shell
CONFIGURATION="
extra-experimental-features = nix-command flakes
extra-trusted-users = ${USER}
"
sh <(curl -L https://nixos.org/nix/install) \
    --no-channel-add \
    --nix-extra-conf-file <(<<< "${CONFIGURATION}")
#+end_src

** Update dependencies

#+begin_src shell
# Update server nixpkgs independently
cd infra && nix flake update nixpkgs

# Update workstation nixpkgs independently
cd workstations && nix flake update nixpkgs

# Update all inputs for a sub-flake
cd infra && nix flake update
cd workstations && nix flake update
#+end_src

** Setup on machine
#+begin_src shell
# On MacOS machine
# - first time
git clone git@github.com:narinari/nix-config.git
nix run nix-darwin -- switch --flake ~/nix-config
# - second and more
darwin-rebuild switch --flake ~/nix-config

# On Linux machine
nixos-rebuild switch --flake .#khali
#+end_src

*** on NixOS
#+begin_src shell
# build and switch (from repo root)
sudo nixos-rebuild switch --flake ~/nix-config#rin

# or build directly from sub-flake
nix build ./infra#nixosConfigurations.rin.config.system.build.toplevel

# build LXC images
nix build ./infra#packages.x86_64-linux.rin-lxc
nix build ./infra#packages.x86_64-linux.jellyfin-lxc
nix build ./infra#packages.x86_64-linux.openclaw-lxc
#+end_src

*** Nix on Ubuntu

#+begin_src shell
# install nix

# enable flake
mkdir -p ~/.config/nix
echo "experimental-features = nix-command flakes" >> ~/.config/nix/nix.conf
# install home-manager
nix run home-manager/master -- init --switch

# upload keys
## on localhost
ssh <remote> sh -c 'mkdir -p ~/.config/sops/age'
scp ~/.config/sops/age/keys.txt <remote>:.config/sops/age # upload age secret keys

# checkout nix-config
## on remote

# build
home-manager switch --flake ~/nix-config"#narinari@work-ec2"
#+end_src

*** Nix on Darwin
#+begin_src shell
# build and switch
darwin-rebuild switch --flake ~/nix-config#hail-mary
#+end_src

** Deploy (remote servers)
#+begin_src shell
# Deploy a single server
deploy .#jarvis

# Dry-run
deploy .#jarvis --dry-activate

# Available deploy targets: jarvis, friday, jellyfin, rin, silk
#+end_src

** Edit secrets

#+begin_src shell
cd nix-secrets
# edit secrets
sops work/xxx.yaml
git ci -m "Update secrets" && git pu

cd nix-config
# update secrets for servers
cd infra && nix flake update my-secrets
# update secrets for workstations
cd workstations && nix flake update my-secrets
#+end_src
